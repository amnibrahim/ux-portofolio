Sports Venue Database

Problem

Yogyakarta is known for its students population. Many of them are aged in 20s. It is common to see university students do some sports between morning and evening. However, in some cases it is hard to find which type of sport and where the sports venue that is available at particular time. In this project, a web page was developed that could accommodate people to look up for sports venue based on their name, address and type of sport.

Role

Front end developer - built a project in React.js

Process

First, a list of sports venue was gathered. The list consisted of different type of sport such as basketball, swimming, futsal, etc. Different kinds of data were also retrieved for example: venue name, address, operating hour and contact. The dataset was taken from previous university class project.

Theere are three components in page: search, result and bookmark component. These three components are located in the same page

Outcome

The most important feature in this case is the search component where everytime user type something in the input element, each character will change the result immediately. The purpose is to make it easier during the searching process by removing a process of clicking button.

The second component which is result component, there are many sports venue result displayed in different color according the type of the sport so that user won't lost track during scanning the result.

The last component is bookmarking. In this case, user may bookmark any result by clicking one of the result which then it will appear on top right of the page. The user also may able to remove the bookmarked result

Tools

React.js, SASS, HTML/CSS/JavaScript
